# Squad: Cura Ancestral 🌿

Bem-vindo ao esquadrão de desenvolvimento do projeto **Manual da Cura Ancestral**.

## 📖 Visão Geral
O projeto visa criar uma solução completa para pessoas de 55 a 75 anos que buscam autonomia de saúde através de ervas comuns e medicina natural, reduzindo custos farmacêuticos e recuperando a qualidade de vida. O coração do método é o **Sistema de Ativação Bio-Molecular (SAB)**.

## 🚀 Estrutura do Projeto
- `data/`: Documentos estratégicos e copy.
    - `persona-dirce.md`: Estudo detalhado da persona "Matriarca da Cura".
    - `estudo-oferta.md`: Documento mestre com o value stack e mecanismo único.
- `funnel/`: Estratégia de vendas.
    - `index.html`: Landing Page / Advertorial (Pronto).

- `members-area/`: Experiência do aluno (Em estruturação).

- `imagem/`: Mockups e ativos visuais gerados.
